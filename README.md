# ⬡ Nexus — Your All-in-One Universe

> Built by [WebCraft Studio](https://webcraft-studioo.netlify.app)
> "Your Vision. Engineered to Perfection."

Nexus is a free, open-source Progressive Web App with 19 built-in tools, AI chat, offline support, 5 themes, and 24+ languages. Installable on any device — iPhone, Android, Windows, Mac, iPad.

![Nexus](public/screenshots/desktop.png)

---

## Features

- AI Chat (OpenRouter + Gemini), AI Art
- Calculator, Timer, Translator, Dictionary
- Health Tracker, Budget, Mood, Habits
- Journal, Recipes, Music Player
- Games, News & Weather, QR Scanner
- Flashlight, Compass, Study Hub
- 5 switchable themes
- 24+ languages
- Full offline support
- Auto-update with changelog
- Anonymous analytics

---

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- Supabase
- vite-plugin-pwa + Workbox
